no terminal do Linux, executar o comando ./agenda para executar o programa.
Opção 2: Caso houver falhas com a linkedição do arquivo Makefile, compilar com gcc/clang separadamente o arquivo contido dentro do diretório ALTERNATIVE


ORGANIZAÇÃO DO CÓDIGO FONTE:
agenda.c == arquivo contendo o main e funções especificas do programa.
data_structure.c == contém a implementação de funções da agenda
data_structure.h == contém declaração de structs e prótotipos das funções referente ao arquivo anterior
sort_algo.c == contém implementação de funções de ordenação em lista e fila
sort_algo.h == prótotipo das funções do arquivo anterior
